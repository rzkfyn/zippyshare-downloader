# zippyshare-downloader
> Sebuah aplikasi web sederhana untuk mendownload file dari zippyshare. (Iklan di zippyshare berlapis lapis njirr).  

## Dibuat Dengan Menggunakan :  

### Back-end : 
* NodeJs
* ExpressJs

### Front-end : 
* Ejs
* Tailwindcss3
